scala-zuul-spring-boot-example
====================
A Scala example for spring-boot and zuul

## Build and Run:

### Build

Build using 
```
./gradlew build
```

### Run

#### Run with gradle 
 
```
./gradlew bootRun
```

#### Run with java 
 
```
java -jar build/libs/scala-zuul-spring-boot-example-0.0.1.jar
```
